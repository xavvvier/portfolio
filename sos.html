## sos ##

SOS is a ASP.NET MVC application to manage the occupational health exams carried out by Laboratorio Colcan to employees of several associated companies.

The application allows the booking of multiple appointments for a single patient, using SignalR to notify in real time the system's users about the assignation preventing the assignation of the same appointment blocks:

It provides custom controls with the help of bootstrap to create autocompletion fields that promote user experience:

User administrators have a granular configuration of authorization based on users or groups:

Between the several configuration options of the application, custom controls such as the Audiogram have been created to allow the input of non-common structured data:
